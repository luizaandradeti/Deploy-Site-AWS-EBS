## Start deploy 

**PowerShell commands - https://nodejs.org/en/download**

````ps1
# Download and install fnm:
winget install Schniz.fnm

# Download and install Node.js:
fnm install 22

# Verify the Node.js version:
node -v # Should print "".

# Verify npm version:
npm -v # Should print "".

````
